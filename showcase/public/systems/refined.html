Design System

Studio

A refined, minimal design system focused on exceptional typography, generous spacing, and sophisticated details.

Philosophy

Studio emphasizes clarity through minimalism. Every element serves a purpose. White space is intentional. Typography leads the hierarchy. Interactions are subtle but meaningful.

Refined

Carefully considered design decisions create a polished, professional aesthetic.

Accessible

WCAG 2.2 AA compliant with semantic HTML and full keyboard support.

Elegant

Subtle animations and micro-interactions enhance the experience.

Typography

Typography is the foundation of Studio. We use Fraunces for headings—a contemporary serif with personality—and Inter for body text, providing clarity and excellent readability.

Type Scale

Name Size Usage
--ds-font-size-xs 12px Captions, labels

Typography Pairing: Fraunces (serif) for headings provides elegance, while Inter (sans-serif) for body text ensures clarity and readability.
